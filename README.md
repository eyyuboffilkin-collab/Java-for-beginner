# Java Beginner Projects

> Fun and beginner-friendly Java projects to learn coding step by step

## Description
This repository contains simple Java projects designed for beginners who want to practice coding, understand Java basics, and explore object-oriented programming (OOP). Each project is small, fun, and focuses on a specific concept.

## Projects Included

1. **HelloWorld** – Learn basic Java syntax and printing output.
2. **Calculator** – Simple console calculator to practice operators and user input.
3. **GuessNumberGame** – Number guessing game to practice loops and conditionals.
4. **TicTacToe** – Classic Tic Tac Toe game to understand arrays and game logic.
5. **SnakeGame** *(optional)* – Build a simple Snake game using Java Swing.
